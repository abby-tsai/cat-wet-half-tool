## 貓食配方所 Vue3.js簡易版

網站：<a href="https://abby-tsai.github.io/cat-wet-half-tool/">Vue3.js 簡易版</a>

這是<a href="https://github.com/abby-tsai/cat-food-tool" target="_blank">貓食配方所</a>作品的簡易版，主要是使用 Vue3.js 框架開發，作為前端框架的磨練，提升開發速度。

共有六個計算功能，分別為：

1. 每日營養計算
2. 濕食乾物比計算
3. 乾糧乾物比計算
4. 全濕食分配
5. 半濕食分配

額外功能：

1. 可儲存計算結果，方便往後比對
2. 可刪除儲存清單
